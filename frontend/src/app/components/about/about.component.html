About

This app allowing users to submit PayForBlob Transactions. It call the PFB API in order to submit a PFB transaction,

To successfully call the API via this app you have to run your node with options: --gateway, --gateway.addr string, and --gateway.port string, to open the gateway and allow anyone to use your IP as an endpoint to submit PFBs.

NODE! This is just for test purpose

Form parameters explanation:

  • scheme - the protocol http or https (default is http)
  • Your Node IP - your server IP
  • Port - gateway port
  • Message - Any text which will be automatically converted to hex-encoded bytes
  • Gas limit - is the limit of gas to use for the transaction
  • Namespace ID - should be 8 bytes.
